PARTE 2 — ARCHITETTURA
I 5 Agenti Specializzati
AGENTE 01
Transaction Analysis
Rileva pattern AML: fan-out, fan-in, circular flow, smurfing. Analizza la rete di destinatari.
→ patterns_detected
AGENTE 02
Risk Assessment
Calcola risk score 0–1 pesando 5 fattori: volume, frequenza, pattern, valori alti, età conto.
→ risk_score, risk_level
AGENTE 03
Fraud Detection
Anomalie temporali, picchi di attività inusuali, storia di laundering dal dataset.
→ fraud_signals
AGENTE 04
Recommendation
Genera azione operativa con priorità: ALERT, REVIEW, BLOCK o MONITOR.
→ recommended_actions
AGENTE 05
Explanation (Granite)
IBM watsonx.ai Granite traduce i risultati tecnici in linguaggio naturale per il compliance officer.
→ explanation (NL)
Flusso: Orchestrate → MCP Server → assessRisk → detectFraud ‖ analyzeTransaction → recommendActions → explainRisk → Report
