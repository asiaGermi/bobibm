### This version introduces the following updates to the datasets. The table lists the dataset name, the columns added or modified, and the corresponding changes:

| Dataset Name    | Column Name            | Reason                                                                                                                                                                                                                                                                                                      |
|:----------------|:-----------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| cards           | Entity_ID              | Previously, this column was not present, to support refrential integrity and to map columns of this dataset with other datatsets this has been added, This column is an unique id to the person holding the card and is as same in users.csv                                                                |
| trans           | Entity_ID              | Previously, this column was not present, to support refrential integrity and to map columns of this dataset with other datatsets this has been added, This column is an unique id to the person making a transaction and is as same in users.csv                                                            |
|                 | Transaction_Date       | All these three columns have been merged into one column to improve readability and consistency                                                                                                                                                                                                             |
|                 | Flight_Date            | All these three columns have been merged into one column to improve readability and consistency                                                                                                                                                                                                             |
| users           | Entity_ID              | Previously, the Person_Index column contained values like 0, 1, 2..., which did not look realistic when mapped with other datasets like homeowners insurance datasets. This new column has now been added to use alphanumeric values, making it more realistic while also supporting referential integrity. |
| _card_trans.csv | _card_trans-accts.csv  | Transactions ordered by card account, then time.                                                                                                                                                                                                                                                            |
|                 | _card_trans-chrono.csv | Transactions ordered by time.                                                                                                                                                                                                                                                                               |
| _cash_trans.csv | _cash_trans-accts.csv  | Transactions ordered by cash owner, then time.                                                                                                                                                                                                                                                              |
|                 | _cash_trans-chrono.csv | Transactions ordered by time.                                                                                                                                                                                                                                                                               |
| All             | All                    | In the previous datasets, there were spaces in between the column names of the datasets, these spaces caused issues when the datasets were uploaded to databases like Db2. This issues has been resolved by adding underscores in column names.                                                             |
